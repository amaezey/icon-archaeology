Introduction

January 2002
Little Icons Vol. 3 by Gary Gehiere

This is the 3rd in the 'Little Icons' series, and I think it's the best yet.

This set was totally finished almost 6 months ago, but at the time I was doing some freelance icon design and I didn't want to release these until I was sure I wouldn't need any of them for that project.

This will probably be the last in the series, but I thought the same thing about 'Little Icons' vol. 2.
