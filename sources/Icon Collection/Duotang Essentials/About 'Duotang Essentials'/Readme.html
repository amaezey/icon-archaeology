Introduction

March 2002
Duotang Essentials by Gary Gehiere

'Duotang' is only my second attempt at a complete system replacement set, ever.

With this design my goal will be to make-over not just OSX but also OS9, the latter of which has substantially more icons than it's roman-enumerated offspring and therefore presents many more demands on me.

I've tried not to appropriate any trademarks or anything of the like, partly because I was toying with the idea of submitting this as a 'Pixelpalooza' entry and also because I've heard that Apple is getting pretty sticky about their trademarks.

When they're done, other 'duotang' sets will be released - all the icons for a complete system folder, a complete set of document icons and of course lots of extra folders. Some of these are done, save a few crucial bits... er, pixels.

Because I should give credit where it's due: 'duotang' is entirely inspired by the Iconfactory's 'World of Aqua' series. However, this is all original work. The 'Butlers' icon is a re-work of the 'Assistants' application icon from OS9/ OSX, which belongs to Apple Computers Inc.
Disclaimer
 

Under OSX you have to hack your system in order to replace the default system icons, which can be dangerous. I can't tell you how to do and I strongly recommend against it.
